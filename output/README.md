# Output Directory

This directory stores analysis results from each run. Each run creates a timestamped folder:

```
output/
└── run-YYYYMMDD_HHMMSS_RANDOM/
    ├── results.json          # Complete analysis results
    ├── metadata.json         # Run statistics and metadata
    ├── session.pkl          # Session data with embeddings (for web UI)
    ├── logs/                # Run-specific logs
    └── visualization/       # Generated HTML visualizations
        └── results.html
```

## Important Notes

- **Required for Web UI**: The web UI loads analysis sessions from this directory
- **Not in Git**: Output files are excluded from version control (too large)
- **Generated by Analysis**: Run `python analyze.py dataset.csv` to create your first session
- **Reusable Sessions**: Each session includes embeddings for fast semantic search

## File Sizes

Typical sizes for ~4,000 comments:
- `results.json`: ~50-60 MB
- `session.pkl`: ~80-100 MB (includes embeddings)
- `metadata.json`: <1 KB

## Cleanup

To remove old runs:
```bash
# Remove specific run
rm -rf output/run-YYYYMMDD_HHMMSS_RANDOM/

# Keep only the 5 most recent runs
ls -t output/ | tail -n +6 | xargs -I {} rm -rf output/{}
```
